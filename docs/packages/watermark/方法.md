# 方法

## API 方法 (Methods)
@purekit/watermark 采用单例模式设计，核心交互主要通过 Watermark.apply 方法完成。

### 核心方法
Watermark.apply(args, container?)
这是库的唯一入口方法，用于初始化、更新或重置水印。它支持多态参数，可根据传入参数的类型自动判断行为。

**类型签名 (Type Signatures)**

```typescript
// 1. 简单文本模式
   Watermark.apply(text: string, container?: string | HTMLElement): this

// 2. 多行文本模式
Watermark.apply(texts: string[], container?: string | HTMLElement): this

// 3. 完整配置模式 (推荐)
Watermark.apply(options: WatermarkOptions): this
```

| 参数名 | 类型 | 必填 | 说明 |
|------|------|------|------|
| args | `string \| any[] \| object` | ✅ | 水印的内容（字符串 / 数组）或完整的配置对象。 |
| container | `string \| HTMLElement` | ❌ | 指定挂载的容器（CSS 选择器或 DOM 元素）。默认为 `document.body`。 |
