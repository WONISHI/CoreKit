# 安装`@purekit/watermark`

@purekit/watermark 是一个轻量级、支持防篡改的前端水印组件，适用于 Vue 2、Vue 3 及原生 JavaScript 项目。

## npm

```shell
  npm install @purekit/watermark --save
```

## pnpm 

```shell
  pnpm add @purekit/watermark
```

## Yarn

```shell
  yarn add @purekit/watermark
```

# 快速开始 (Quick Start)

## 在 Vue 3 / Vue 2 中使用
建议在组件挂载完成后（onMounted 或 mounted）调用，以确保 DOM 元素已存在。

```shell
import Watermark from '@purekit/watermark';

// Vue 3 Composition API
import { onMounted, onUnmounted } from 'vue';

onMounted(() => {
  Watermark.apply({
    content: ['内部资料', '严禁外传'],
    fontColor: 'rgba(200, 0, 0, 0.15)',
    zIndex: 9999
  });
});

// 销毁组件时（如果需要手动清理，虽然后续渲染会覆盖，但推荐清理）
onUnmounted(() => {
   // 如果库提供了 destroy 方法，在此调用
});
```

## 原生 JS 使用

```shell
import Watermark from '@purekit/watermark';

// 确保 DOM 加载完毕
document.addEventListener('DOMContentLoaded', () => {
  Watermark.apply('我的水印内容');
});
```
